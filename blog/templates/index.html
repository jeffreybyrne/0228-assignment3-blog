What is a client?

A client is a program that someone can use to access the internet, such as Google Chrome or Mozilla Firefox.

What is a server?

A server is a computer that's connected to the internet and can serve up assets that are requested by a client.

What is an HTTP request?

An HTTP request is a request from one machine to another over the internet adhering to the specific HTTP protocol. General methods are GET (requesting data), POST (creating data), PATCH (to update data), PUT (to replace data), and DELETE (to delete data).

What is an HTTP response?

An HTTP response is a message from a server back to the client that requested it. The response will generally contain a response code indicating the status of the request, and sometimes a body describing what was done.

What is Django and why do we use it?

Django is a framework that allows use of some Python when creating websites, allowing more interactivity between the backend and the frontend.
